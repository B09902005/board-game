# 自製桌遊 遊戲規則

## 簡介

這款遊戲是由盧冠綸所自製的桌遊。玩家會從起點開始遊戲，勝利條件是成為所有玩家之中，最快到達終點的人。

## 遊戲人數

遊戲人數為 1~4 人。您可以選擇進行多人同樂遊戲，也可以選擇單人模式與 AI 來場激烈對決。

## 遊戲流程

遊戲為回合制，由每位玩家輪流進行，直到有玩家到達終點為止。

每位玩家在輪到自己時的行動流程如下：

### 1. 決定移動方式

玩家在自己的回合，有三種移動方式。玩家可以選擇其中一種方式來移動。

* **骰骰子**：以骰子「隨機決定」自己本回合的行動步數。
* **控骰卡**：以控骰卡「直接決定」自己本回合的行動步數。
* **傳送門**：付出一定金錢，直接傳送至指定地點。（付出的金錢數量與地點詳見遊戲）

### 2. 決定移動位置

如果玩家選擇骰骰子或使用控骰卡，得知移動步數後，可以根據步數來選擇要移動到哪一格。移動方向並無任何限制，但在移動過程中不能走回頭路。此外，移動步數必須要剛好用完，不能多走或少走。

在真人遊玩時，可以藉由直接點擊地圖上的旗子，來選擇自己要移動到哪一格。

### 3. 觸發格子效果

玩家移動完後，會觸發該格子的效果。以下是遊戲中會出現的各種格子與觸發效果。

* ![](https://hackmd.io/_uploads/Bk93YnZya.png) **普通格**：不會發生任何事。
* ![](https://hackmd.io/_uploads/SJ7ZY2-1p.png) **地點格**：不會發生任何事。如果踩到終點就贏得遊戲。
* ![](https://hackmd.io/_uploads/rkoF_2bkT.png =6%x) **金錢格**：玩家獲得 30 元。
* ![](https://hackmd.io/_uploads/ByUNcnZ1p.png =6%x) **命運格**：玩家抽一張命運卡，並在下面兩者之中選擇一個來執行。
    * 直接執行卡片效果。
    * 付出一定金錢，將卡片嫁禍給他人來執行（價格由卡片內容而定）。
* ![](https://hackmd.io/_uploads/BJTHq2-ya.png =6%x) **道具格**：玩家獲得一張道具卡。道具卡有以下三種：
    * 嫁禍無效卡（一次性）：在被嫁禍命運卡時，可以使用這張卡來抵擋，該命運卡將不會被執行於任何人。
    * 控骰卡（一次性）：可以直接決定該回合的行動步數。
    * 永久效果卡（永久）：獲得某些神秘的強化效果。

## 提示

這個遊戲會牽涉到不少的策略、運氣等因素，每種策略都會為遊戲性帶來不同的效果，也會導致不同的結果。本遊戲的設計中，並沒有所謂的「無敵」的策略，但玩家還是可以去探索各種不同的策略或玩法，來讓自己贏得勝利。這裡提供了幾種可行的策略供玩家參考，也歡迎大家多多探索各種不同的玩法。

* **速攻型玩法**：追求到終點的最短距離，好讓自己跑得比其他人快來贏得勝利。
* **傳送型玩法**：追求大量金錢，好讓自己可以利用傳送門來贏得勝利。
* **道具型玩法**：追求大量道具，好讓自己能抵禦他人嫁禍，並使用控骰卡來贏得勝利。
* **爆發型玩法**：追求變數大的命運格，好讓自己能一夕致富來贏得勝利。
* **平均型玩法**：讓自己的各項數值保持平均，好讓自己能夠穩定贏得勝利。
